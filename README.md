# Comprobador de Trazabilidad

Aplicación desarrollada en Python para comprobar si una trazabilidad leída mediante un lector QR ya existe en una base de datos de Excel.

## Funcionalidades

- Cargar una base de datos desde un archivo Excel.
- Comprobar automáticamente cada código leído.
- Mostrar si la pieza es correcta o duplicada.
- Guardar un registro de los escaneos realizados.

## Tecnologías

- Python
- Tkinter
- Pandas
- OpenPyXL
